# vim-autofold

Folds the entire file, but opens just enough to make the current line not folded.

## Usage

| Command            | Description         |
| ------------------ | ------------------- |
| `:Autofold`        | Toggle autofolding  |
| `:AutofoldEnable`  | Enable autofolding  |
| `:AutofoldDisable` | Disable autofolding |
