<h1 align='center'>
vim-gitgrep
</h1>

<p align='center'>
Searches the project using <code>:GG &lt;search&gt;</code>.
</p>

![Screencast](images/preview.gif)

## Install

Install using vim-plug:
```vim
Plug 'rstacruz/vim-gitgrep'
```

Search using `:GG`:

```
:GG hello
```

Read the [documentation](./doc/gitgrep.txt):

```vim
:help gitgrep
```
